# مهمة حصر بصري شامل (تقييم تقني — ليست مسرح جريمة)
هذه صورة تقييم من مجموعة اختبار معيارية. المطلوب **حصر كل نسخة ظاهرة** من الأصناف الستة التالية فقط، حتى لو بدت اعتيادية تماماً — قاعدة «الأغراض الاعتيادية ليست أدلة» **لا تنطبق** في هذه المهمة:

| الصنف | ابدأ `name_ar` بـ | `category` |
|---|---|---|
| سكين | «سكين» | `weapons` |
| مقص | «مقص» | `weapons` |
| هاتف محمول | «هاتف محمول» | `documents_devices` |
| حاسوب محمول | «حاسوب محمول» | `documents_devices` |
| زجاجة/قارورة | «زجاجة» | `trace` |
| كتاب | «كتاب» | `documents_devices` |

القواعد:
- اكتشاف مستقل لكل نسخة (ثلاث زجاجات = ثلاثة اكتشافات) مع `local_id` تسلسلي.
- `bbox_2d` بإحكام حول كل نسخة: أعداد صحيحة 0–1000 نسبةً لأبعاد الصورة، تُقاس من الصورة الفعلية.
- لا تذكر أي صنف خارج الستة إطلاقاً، ولا تختلق نسخاً غير ظاهرة؛ الجزء المحجوب جزئياً يُحصر إن كان تمييزه مؤكداً.
- `confidence` بحسب وضوح النسخة. بقية الحقول النصية: وصف موجز جداً أو "".
- إن لم يظهر أي صنف من الستة أرجِع قائمة فارغة.

أجب بصيغة JSON فقط وفق مخطط `DetectionResult`.

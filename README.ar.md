# أثر — نظام تحليل مسرح الجريمة بالذكاء الاصطناعي

> **«كل تماسٍ يترك أثراً» — مبدأ لوكار في تبادل الآثار**

نظام تحليل أدلة جنائية موجَّه للاستخدام القضائي في المملكة العربية السعودية. يرفع المحقق الصور ومقاطع الفيديو من مسرح الجريمة، فيحللها نموذج **Qwen3-VL** ويُنتج النظام **جدولاً زمنياً للأدلة**، و**معرضاً مصوَّراً معلَّماً**، و**تقريراً تحليلياً شاملاً بالعربية الفصحى القانونية**، مع تصدير **PDF/A** و**DOCX**، و**سلسلة حيازة** ببصمات SHA-256، و**سجل تدقيق** مسلسل البصمات لا يقبل التعديل الخفي. الواجهة عربية بالكامل، من اليمين إلى اليسار.

## ⚠️ الحدود القانونية والأخلاقية

كل مُخرَج يحمل الوسم: **«تحليل بمساعدة الذكاء الاصطناعي — يتطلب تحقيق خبير مؤهل قبل أي استخدام قانوني»**. النظام لا يُصدر أحكاماً قانونية أبداً: لا إدانة، ولا تحديد سبب وفاة، ولا استنتاج نيّة. وكل اكتشاف تقل درجة ثقته عن الحد (٧٥٪ افتراضاً) يُعلَّم إلزامياً **«يتطلب مراجعة بشرية»** ويدخل طابور المراجعة. اكتشافات «الوجود البشري» توصَف وصفاً موضعياً فقط — بلا تحديد هوية — وتُموَّه تلقائياً في النسخ المعلَّمة.

## التشغيل بأمر واحد

<div dir="ltr">

```bash
docker compose up --build
```

</div>

ثم افتح **http://localhost:8090**. بدون ملف `‎.env` يعمل النظام في **وضع المحاكاة**: لا مفتاح ولا معالج رسومي — وخط الأنابيب كاملاً (رفع ← تحليل ← جدول زمني ← مراجعة ← تقرير PDF/A) يعمل على العينات المرفقة.

لاستخدام النموذج الفعلي: انسخ `‎.env.example` إلى `‎.env` واضبط `MODEL_MODE=api` مع مفتاح OpenRouter، أو `MODEL_MODE=local` للاستدلال المحلي المعزول عبر vLLM (الملف الشخصي `gpu` في docker compose).

> **سيادة البيانات:** وضع `api` يرسل صور المسرح إلى مزوِّد سحابي. للقضايا الحقيقية استخدم الوضع المحلي المعزول تماماً.

## خريطة المستودع

| المسار | الغرض |
|---|---|
| `backend/app/prompts/` | موجِّهات النظام العربية (منع الاختلاق، صد حقن التعليمات، تعريف الفئات) |
| `backend/app/pipeline/stages/` | مراحل خط الأنابيب العشر |
| `backend/app/modelclient/` | عميل Qwen3-VL الموحَّد (سحابي/محلي/محاكاة) مع إصلاح JSON وحد التكلفة |
| `frontend/DESIGN.md` | نظام التصميم — مُكيَّف للعربية RTL من `DesignMD.txt` |
| `samples/` | عينات تمثيلية بريئة + بيانات المحاكاة + شبكة معايرة الإحداثيات |
